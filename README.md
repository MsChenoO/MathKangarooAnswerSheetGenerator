# Math Kangaroo Answer Sheet Generator

A modern React application that generates fully customizable Math Kangaroo practice answer sheets

## Features

- Generate customized answer sheets for grades 1-12 student 
- Automatic question count adjustment (24 for grades 1-4, 30 for grades 5-12), like the real exam.
- Can be download as PDF, PNG, or Print directly
- Auto-generated mock Test IDs and barcodes, so that the mock feels real.
